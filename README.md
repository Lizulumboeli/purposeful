# ✨ Purposeful — AI Mentor for Women in Tech

> A command-line AI mentor powered by Claude (Anthropic) — built to help women in tech own their careers in AI & product.

## Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | 🗺️ Career Path Generator | Maps out realistic career trajectories with timelines |
| 2 | 📄 Resume Optimizer | Rewrites your resume for AI PM roles with ATS keywords |
| 3 | 🎯 Job Matcher | Finds your best-fit roles based on your background |
| 4 | 📚 Learning Path Generator | Personalized 30/60/90-day learning plans |
| 5 | 🎤 Interview Prep | Realistic mock interviews with detailed feedback |
| 6 | 📰 AI News & Trends | Stay current on what matters for your career |
| 7 | 🎭 Roleplay Scenarios | Practice salary negotiation, pitches, tough conversations |

## Setup

### 1. Get a Claude API key
Sign up at [console.anthropic.com](https://console.anthropic.com) and create an API key.

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Add your API key
Create a `.env` file in this folder:
```
ANTHROPIC_API_KEY=your_key_here
```

### 4. Run Purposeful
```bash
python purposeful.py
```
